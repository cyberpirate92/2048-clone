# 2048 ![](https://github.com/cyberpirate92/2048-clone/workflows/Tests/badge.svg)

A 2048 clone built using Typescript and HTML5 Canvas.

## Running locally
1. Clone project `git clone https://github.com/cyberpirate92/2048-clone
2. Install dependencies `npm install`
3. Build project `npm run build`
4. Open index.html page in a browser

## Running tests
To run the unit tests, use `npm test`
